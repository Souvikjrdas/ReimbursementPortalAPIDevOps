---------------------------------Welcome To My Reimbursement Portal--------------------------------------

Steps to land on to th website

i)  Open the Reimbursement portal solution file in Visual Studio.

ii) Add WebAPI as the startup project and run it with IIS Express.

iii) After launching the server minimize the Visual Studio go to
  a) User Presentation folder.
  b) Go to Frontend_Portal and then open it with Visual Studio Code.
  c) make sure you go to Frontend_Portal by running "cd Frontend_portal". 	
  d) Try to launch the node modules.
  e) After successful installation of node modules create a new terminal and run "ng serve" command.
   
  launch the URL provided by the terminal on any browser.

 -------------------------------------YOU ARE ALL SET TO GO------------------------------------------------
